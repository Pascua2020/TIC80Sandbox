#### 🎆-Trompos

-Jugador 1

#### 🎆-Vida

Cada uno empieza con 100.

#### 🎆-Circulos

Agregan ( Circulo Azul ) o quitan ( Circulo Rojo ) vida al estar en ellos.

#### 🎆-Tiempo

Desde 30 segundos para abajo,gana el que llega al final con más puntos de vida ( si es que ambos sobreviven ).

#### 🎆-Rap

Rap del Trompo.
